1. MongoDB URI olish:
   - mongodb.com/cloud/atlas ga kiring
   - Cluster → Connect → Drivers
   - Connection string ni nusxalang

2. .env faylini oching va MONGO_URI ni o'rnating:
   MONGO_URI=mongodb+srv://...

3. Kutubxonalarni o'rnating:
   npm install

4. Serverni ishga tushiring:
   node server.js

5. Brauzerda oching:
   http://localhost:3000
